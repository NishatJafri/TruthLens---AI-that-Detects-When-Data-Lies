#TruthLens: AI that Detects When Data Lies

Overview

TruthLens is an AI-driven system designed to analyze datasets and detect misleading, manipulated, or suspicious data patterns. In today’s digital world, statistics and graphs are often presented in ways that mislead people. TruthLens helps users verify the credibility of data using machine learning techniques and interactive data visualization.
The system analyzes datasets, identifies anomalies, and generates a credibility metric called the Truth Score.

Problem Statement

Many reports, surveys, and business dashboards present data in misleading ways, such as exaggerated percentages, incomplete datasets, or manipulated visualizations. This makes it difficult for users to determine whether the information is reliable.

TruthLens provides an automated solution that analyzes datasets, identifies suspicious patterns, and visualizes insights through an interactive dashboard.

Objectives

- Detect anomalies and unusual patterns in datasets.
- Identify potentially misleading or manipulated data.
- Generate a Truth Score representing the reliability of the dataset.
- Provide interactive data visualization through Power BI dashboards.

Key Features

- Upload dataset (CSV / Excel)
- Data cleaning and preprocessing
- Outlier and anomaly detection using machine learning
- Truth Score calculation
- Interactive visualization dashboard using Power BI
- Automated data insights

Technologies Used

1. Python – Data processing and machine learning
2. Pandas & NumPy – Data analysis
3. Scikit-learn – Anomaly detection algorithms
4. Power BI – Interactive dashboard and visualization
5. Matplotlib / Plotly – Data visualization support
6. Streamlit / Flask – User interface for data upload

Role of Power BI in the Project

Power BI will be used to create an interactive dashboard that visualizes the analysis results generated by the system.
The dashboard will display:
- Dataset summary statistics
- Detected anomalies
- Suspicious trends in the data
- Truth Score of the dataset
- Comparative visualizations
These dashboards will help users easily understand whether the dataset is reliable or misleading.

System Workflow

- User uploads a dataset through the interface.
- Python processes and cleans the dataset.
- Machine learning algorithms detect anomalies or suspicious patterns.
- A Truth Score is calculated.
- Processed data is exported for visualization.
- Power BI dashboard displays insights and results.

Example Use Cases

- Checking reliability of business reports
- Detecting manipulated survey data
- Analyzing financial or sales datasets
- Validating research data

Future Enhancements

- Real-time data verification
- Automatic detection of misleading graphs
- Integration with online data sources
- Advanced explainable AI models
